# Flask-API-v1
A simple Python Flask API project to demonstrate backend routing and JSON response handling.

## Tech Stack
- Python 3.14
- Flask 3.1.3

## How to Run
1. Clone the repository.
2. Install Flask: `pip install flask`
3. Run: `python app.py`
4. Access: `http://127.0.0.1:5000/`
